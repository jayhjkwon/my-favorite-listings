## Demo
- [https://my-favorite-listings.herokuapp.com/](https://my-favorite-listings.herokuapp.com/)

## Prerequisites
- Node.js v8.0+

## Quick Start
```sh
npm install
npm run dev
```

## Run test
```sh
npm test
```

## Run in production mode
```sh
npm start
```

## Tech stack used for this app
- React
- Redux
- React-Router
- Styled Components
- Bootstrap
- Jest
- Enzyme
